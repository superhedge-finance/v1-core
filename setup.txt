1. Install Packages:
npm install

2. Create secret.json file with environment variables:
{
    "PRIVATE_KEY" : "xxxxxxxxxxxxxxxxxxxxxyyyyyyyyyyyyyy",
    "API_KEY_ARBISCAN": "zzzzzzzzzzzzzzzzxxxxxxxxxxxxx"
}

3. Deploy contracts:
Edit the file scripts/deploy.js, and run
npx hardhat run scripts/deploy.js --network arb 

 i)     SHTokenFactory
 ii)    SHFactory

4. Verify each of the deployed contracts by running:
npx hardhat verify --contract "contracts/[CONTRACT.sol]:CONTRACT" --network arbitrum 0x_CONTRACT_ADDRESS
